# 13. 버전 확인 (CycleCloud · Slurm)

운영 지원 시 CycleCloud Server 및 Slurm의 버전 정보를 확인하는 방법입니다.

---

## 13.1 CycleCloud Server 버전

포털 우측 상단의 **`?` (도움말)** 아이콘에서 확인합니다.

![CycleCloud 버전 확인](images/add-node/1780364891805.png)

서버 VM에서 CLI로도 확인할 수 있습니다.

```bash
cyclecloud --version
```

---

## 13.2 Slurm 버전

**Clusters → 해당 클러스터 선택 → Edit → Advanced Settings** 의 **Slurm Settings** 항목에서 지정된 Slurm 버전을 확인합니다.

![Slurm 버전 확인](images/check-version/1780365604255.png)

스케줄러 노드에 접속해 직접 확인할 수도 있습니다.

```bash
sinfo --version
```
