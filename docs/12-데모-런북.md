# 12. 데모 런북 (진행자용 체크리스트)

MSP 대상 데모/교육 진행자가 순서대로 수행하는 체크리스트입니다.

---

## 사전 준비 (데모 시작 5분 전)

- [ ] Azure CLI 로그인 및 올바른 구독 선택 확인
      ```bash
      az login
      az account show -o table
      ```
- [ ] 서버 VM 기동:
      ```bash
      az vm start -g rg-cyclecloud-training -n cc-server
      ```
- [ ] 1~2분 대기 후 서비스 상태 확인:
      ```bash
      az vm run-command invoke -g rg-cyclecloud-training -n cc-server \
        --command-id RunShellScript --scripts "systemctl is-active cycle_server" \
        --query "value[0].message" -o tsv
      ```
- [ ] 웹 포털 접속 확인: `https://cc-cyclecloud-ekwphu.koreacentral.cloudapp.azure.com`

## 데모 종료 후 (⚠️ 비용 관리 정리 필수)

- [ ] 데모 클러스터 중지:
      ```bash
      cyclecloud terminate_cluster slurm-demo
      ```
- [ ] 서버 VM 일시 중지 (비용 최소화):
      ```powershell
      .\infra\destroy-lab.ps1 -Mode Deallocate
      ```
- [ ] (필요시) 리소스 그룹 전체 삭제:
      ```powershell
      .\infra\destroy-lab.ps1 -Mode Delete
      ```
